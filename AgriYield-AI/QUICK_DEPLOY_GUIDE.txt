IF RENDER BUILD FAILS:
=====================

Common issues & fixes:

1. "ModuleNotFoundError: gunicorn"
   → Make sure gunicorn is in requirements.txt ✓

2. "No such file or directory: crop_yield_predict"
   → Check path in Start Command
   → Should be: cd crop_yield_predict

3. "Static files not found"
   → Build command includes collectstatic ✓

4. Port binding error
   → Make sure Start Command has: --bind 0.0.0.0:$PORT
