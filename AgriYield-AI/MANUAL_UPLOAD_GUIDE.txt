=====================================
WHAT IF BUILD FAILS?
=====================================

Common errors & fixes:

❌ "Error: No module named 'gunicorn'"
✅ Solution: Gunicorn is already in requirements.txt ✓

❌ "ModuleNotFoundError: No module named 'crop_yield_predict'"
✅ Solution: Build command has cd crop_yield_predict ✓

❌ "Static files not found (CSS/JS broken)"
✅ Solution: Build command includes collectstatic ✓

❌ "Connection refused on port"
✅ Solution: Start command uses $PORT variable ✓

If still failing:
1. Check Render logs for exact error
2. Search error message in docs
3. Contact Render support (they're helpful!)
